Namespace: validate

Thor. validate

This is the Thor's validation module, used to validate forms.
For using you just need to use THOR ou T keyword, 'validate' for this module, and call a method. Like:

if(T.validate.cpf('00000000000'))
    alert("CPF is valid!");
